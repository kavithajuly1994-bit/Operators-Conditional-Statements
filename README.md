🔢 Conditional Logic Checker (Weird / Not Weird)

📌 Project Overview
* This project is a simple Python program that determines whether a given number is "Weird" or "Not Weird" based on specific conditions. It demonstrates the use of operators and conditional statements to control program flow.

🎯 Objective
* To understand and apply conditional logic in Python
* To classify numbers based on given rules
* To practice if-elif-else statements and operators

⚙️ Tools & Technologies Used
* Python – Core programming language
* Google Colab

🛠️ What I Have Done
* Accepted user input for an integer value
* Applied modulus operator (%) to check if the number is odd or even
* Implemented conditional logic using:
     if
     elif
     else
* Classified numbers based on the following rules:
     Odd → Weird
     Even (2–5) → Not Weird
     Even (6–20) → Weird
     Even (>20) → Not Weird
* Displayed the result based on the evaluation

📊 Sample Output
Enter a number:10
Weird

📚 Conclusion
* This project helps in understanding how conditional statements work in Python. It builds a strong foundation for writing decision-based programs and handling logical conditions effectively.
